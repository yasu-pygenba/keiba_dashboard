## Keiba Dashboard 

競馬予測AIの [成績検証・可視化] を目的とした個人開発プロジェクトです。

予測精度を目的とせず、
「どのような予測ロジックが、どの条件で、どのような成績になるのか」を
継続的に記録・分析するための実験基盤として作成しています。

---

概要 

- Pythonで作成した競馬予測AIの結果をもとに
- レース終了後の成績データをDBに保存
- Django + ORM を用いて集計し、Web画面で可視化します

本リポジトリでは 成績ダッシュボード（v1.0） を公開しています。

---

主な機能（v1.0） 

- レース単位の成績データ管理
- 条件別の成績集計
  - 競馬場
  - 券種（単勝 / 複勝 / 馬連 / 馬単 / ワイド）
  - 印ロジック（印 / 印2 / 印3）
- 表示指標
  - 投資額
  - 払戻額
  - 収支
  - 的中率
  - ROI
